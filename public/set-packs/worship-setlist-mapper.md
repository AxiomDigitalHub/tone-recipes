# Worship Set Pack — Setlist Mapper

**Preset:** FK Worship
**Platform:** Line 6 Helix / HX Stomp / Helix LT
**Updated:** April 2026

## Signal Chain

```
Deluxe Comp → Minotaur (Klon) → Scream 808 → Essex A30 (AC30) →
2x12 Blue Bell → 70s Chorus → Transistor Tape (1/4 note) →
Vintage Digital (dotted 1/8) → Plate Reverb → Glitz Reverb
```

## Snapshot Reference

| # | Snapshot | What It Does | When to Use |
|---|----------|-------------|-------------|
| 1 | **CLEAN** | Pure clean, subtle plate reverb + quarter note delay | Verses, intros, quiet moments |
| 2 | **DRIVE** | Light Klon overdrive, amp pushed slightly | Pre-chorus, building sections |
| 3 | **DRIVE+** | Klon + 808 stacked, medium gain | Choruses, driving sections |
| 4 | **LEAD** | Full drive + solo delay, level boost | Solos, melodic lines, turnarounds |
| 5 | **CLN AMBI** | Clean + chorus + big dotted eighth + shimmer reverb | Ambient pads, intro swells, prayer |
| 6 | **AMB DRV** | Light drive + full ambient effects | Ambient with edge, ethereal drive |
| 7 | **ROCK** | Heavier crunch, AC30 pushed hard | Rock-influenced worship, bridge hits |
| 8 | **SWELLS** | Ultra-clean + max reverb/delay/shimmer | Volume swells, pad-like textures |

## Quick Start Guide

1. Load `FK-Worship.hlx` into HX Edit
2. Set your Helix to **Snapshot mode** (or Snap/Stomp hybrid)
3. Look up today's setlist in the mapper above
4. Start with the **Verse** snapshot for each song
5. Switch snapshots as you move through sections
6. All snapshot switches are **gapless** with trails enabled

## Tips

- **Guitar volume knob** is your friend — roll back to clean up any snapshot further
- **SWELLS** snapshot works best with an expression pedal for volume swells, or use your guitar volume knob
- **CLN AMBI** is your "prayer time" and "between songs" tone
- **ROCK** is for songs like Praise, Battle Belongs, or See A Victory where you need more edge
- The Klon (Minotaur) is the heart of this preset — it adds compression and mid push even at low gain
- If you need more gain, bump the amp Drive in snapshots 6 and 7

## Stomp Mode Suggestions (Hybrid Mode)

If using snapshot/stomp hybrid (4 snapshots on bottom, 4 stomps on top):

| Footswitch | Assignment |
|-----------|-----------|
| FS1 | Snapshot: CLEAN |
| FS2 | Snapshot: DRIVE |
| FS3 | Snapshot: DRIVE+ |
| FS4 | Snapshot: LEAD |
| FS5 | Stomp: Chorus (toggle) |
| FS6 | Stomp: Dotted Eighth Delay (toggle) |
| FS7 | Stomp: Glitz Reverb (toggle) |
| FS8 | Stomp: Scream 808 (toggle) |

This gives you 4 core gain stages on the bottom row, plus individual effect toggles on top for customizing within any snapshot.
